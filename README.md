# Airbnb-Price-Prediction
## 📌 Overview
This project analyzes Airbnb listings and predicts prices using machine learning.

## 📊 Dataset
Inside Airbnb (public dataset)

## ⚙️ Tech Stack
- Python
- Pandas, Scikit-learn
- XGBoost
- Streamlit

## 🚀 Results
- Random Forest MAE: ~XX
- Key drivers: room type, location, number of reviews

pip install -r requirements.txt
